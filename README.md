# Cities 1.0.0

### Requirements

For this app to work it's needed the gems described below:

* mysql2
* Formatador
* Highline
* ruby 2.0.1

### Install & Usage

To install it just clone the repository to the desired location and run:

```ruby
ruby init.rb
```

This was created with the intention to teach to code.
